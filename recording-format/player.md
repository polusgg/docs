
Player Entry format:

|     Type     |      Name      |       Description       |
|--------------|----------------|-------------------------|
| `UInt8`      | Player ID      | The player's ID         |
| `string`     | Player Name    | The player's name       |
| `UInt8`      | Cosmetic Count | The number of cosmetics |
| `Cosmetic[]` | Cosmetics      | Detail about cosmetics  |
